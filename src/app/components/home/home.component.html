Aplicacion web realizada por Jesús Paredes para el ciclo de Desarrollo de Aplicaciones Multiplataforma

Aplicacion realizada para el módulo de DI, hecha por Jesús Paredes en ANGULAR que consta de:

  • Componente Home: representa la entrega inicial de la aplicación.
  • Componente Ciclos: representa tantas cartas como ciclos existan en el CiclosService.
  • Componente Asignaturas: representa tantas cartas como asignaturas haya en el servicio AsignaturaService. Adicionalmente se cuenta con un filtro para poder ver asignaturas específicas.
  • Componente Asignatura: representa el aspecto que tendrá una carta de las de arriba mencionadas.
  • Componente DetalleCiclo: representa el detalle de un ciclo,donde se mostrarán los conocimientos necesarios para poder aprobar el ciclo y todas las asignaturas que lo componen.
  • Componente NavBar: representa la barra de navegación superior donde se muestra un menú con las siguientes posibilidades:
    • Inicio: Llevará al componente Home.
    • Ciclos: Llevará al componente Ciclos.
    • Asignaturas: Llevará al componente Asignaturas.
  • Servicios: se deberá crear al menos dos servicios que provean de datos tanto para los ciclos como para las asignaturas.
  • Pipes: se deberá crear al menos dos directivas personalizada que realice alguna transformación en la aplicación.
